Instructions
CSV Format

Your CSV should include these columns:

  • ID - Item ID (for updates)
  • Title - Item name
  • Description - Item description
  • Purchase Price - Cost paid
  • Status - watch, won, listed, sold
  • Target Resale Price - Expected sale price
How it Works
  • Items with existing IDs will be updated
  • Items without IDs will be created as new
  • Empty fields will be ignored
  • Invalid data will be skipped with warnings
Tip: Download the current inventory as a template to see the exact format.
